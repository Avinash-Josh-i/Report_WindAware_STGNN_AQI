\chapter{PROJECT BUDGET}

The proposed system is mainly a software and research project, so the required budget is relatively low. Most libraries and datasets are open source or publicly available. The main expected expenses are internet access, optional cloud or \gls{gpu} compute, deployment, documentation, and miscellaneous project needs.
