\chapter{REMAINING TASKS}


The following tasks remain to reach the objectives stated in the project proposal:

\begin{enumerate}
    \item Complete the wind-aware dynamic graph construction module (bearing matrix, dynamic edge weights, graph snapshots, sliding windows).
    \item Implement additional baseline models (Random Forest, XGBoost) for comparison against the linear baseline before committing to the full GAT-GRU architecture.
    \item Implement the GAT-GRU forecasting model using PyTorch Geometric, including a graph-aware data loader distinct from the current per-station CSV pipeline.
    \item Implement virtual-node estimation for unmonitored locations, including leave-one-station-out validation.
    \item Implement the explainability module (attention-weight extraction from GAT layers, SHAP-based feature attribution, temporal sensitivity analysis).
    \item Build the interactive dashboard for forecasts, AQI categories, spatial maps, and explanation visualizations.
    \item Extend evaluation to include peak-event detection metrics (precision, recall, F1) in addition to regression metrics.
\end{enumerate}
